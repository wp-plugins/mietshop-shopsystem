Shopsoftware Modul für WordPress. Artikel des Onlineshops per Widget in die Seite integrieren.

== Description ==

Das Plugin ermöglicht die Integration von Artikeln, Bestsellern und Highlight-Produkten aus dem Mietshop Shopsystem. Den Mietshop erhalten Sie 6 Monate kostenlos. Erst dann entscheiden Sie sich für eine der günstigen Varianten. Sie betreiben also einen vollwertigen Onlineshop und können dessen Inhalte (wie Produkte, Besteller-Box, Highlight der Woche Box, usw..) in Ihren Wordpress Blog an beliebiger Stelle einfügen. Somit erreichen Sie den optimalen Content / Produkt-Mix, mit direkter Verlinkung zu Ihrem Onlineshop.

Einfache Integration via Wysiwyg Editor. Einstellmöglichkeiten. Gute Anleitung. Der Mietshop ist binnen weniger Sekunden automatisch installiert (Cloudsystem), ist einfach bedienbar und modular erweiterbar.

Wenn Sie das Plugin mögen, freuen wir uns über ein gutes Rating. Kritik und Anregungen nehmen wir ernst und setzen es umgehen im Plugin um. Updates werden regelmässig zur Verfügung gestellt.

== Installation ==

1. Die neueste Version des Widgets herunterladen.
2. Zur Admin-Seite __Plugins -> Installieren__ gehen und dort den Tab __Hochladen__ wählen.
3. Das ZIP-Paket direkt hochladen.
4. Nach dem Hochladeprozess auf __Aktivieren__ klicken.
