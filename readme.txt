Shopsoftware Modul für WordPress. Artikel des Onlineshops per Widget in die Seite integrieren.

== Description ==

Das kostenlose Plugin ermöglicht die Integration von Artikeln, Bestsellern und Highlight-Produkten aus dem Mietshop Shopsystem. Der Mietshop ermöglicht die einfache Erstellung eines Onlineshop und ist die Voraussetzung für die Nutzung/Funktionsfähigkeit des Wordpress Plugins. Den Mietshop erhalten Sie 6 Monate kostenlos. Erst dann entscheiden Sie sich für eine der günstigen Varianten. Die Inhalte Ihres Mietshops (wie Produkte, Besteller-Box, Highlight der Woche Box, usw..) können Sie in Ihren WordPress Blog an beliebiger Stelle einfügen. Somit erreichen Sie den optimalen Content / Produkt-Mix, mit direkter Verlinkung zu Ihrem Onlineshop.

Einfache Integration via WYSIWYG Editor, viele Einstellmöglichkeiten, gute Anleitung. Der Mietshop ist binnen weniger Sekunden automatisch installiert (Cloudsystem), ist einfach bedienbar und modular erweiterbar: [www.Mietshop.de](http://www.mietshop.de/content/kostenlosen-shop-eroeffnen.html?utm_source=wordpress&utm_medium=plugin&utm_campaign=mietshop-wordpress)

Wenn Sie das Plugin mögen, freuen wir uns über ein gutes Rating und Ihre Weiterempfehlung. Kritik und Anregungen nehmen wir ernst und setzen es umgehen im Plugin um. Updates werden regelmässig zur Verfügung gestellt.

= Bedienung =
1. Um Inhalte einzufügen, gehen Sie zu den Seiten oder Beiträgen. Dort finden Sie einen "Warenkorb" Button in der Werkzeugleiste des visuellen WYSIWYG Editors.
2. Wählen Sie nun im Dropdown aus, welches Widget Sie einfügen möchten (Highlight, Bestseller oder eine Artikelvorschaubox) und geben Sie ggf. eine Artikelnummer ein. Sie können noch wählen, ob es linkbündig oder rechtsbündig eingefügt werden soll.
3. Im Editor sehen Sie nun eine Dummygrafik, die dann im Frontend durch Ihre echten Inhalte aus Ihrem Mietshop dynamisch ersetzt wird.
4. Inhalte können ebenfalls über Widget-Funktion eingefügt werden

== Installation ==

1. Die neueste Version des Widgets herunterladen.
2. Zur Admin-Seite __Plugins -> Installieren__ gehen und dort den Tab __Hochladen__ wählen.
3. Das ZIP-Paket direkt hochladen.
4. Nach dem Hochladeprozess auf __Aktivieren__ klicken.


1. Installieren Sie das Plugin, indem Sie unter __Plugins__ -> __Installieren__ nach "Mietshop Shopsystem" suchen
2. Klicken Sie anschliessend auf __Installieren__
3. Das Plugin installiert sich nun automatisch
4. Klicken Sie anschliessend auf __aktiviere dieses Plugin__
5. Klicken Sie nun auf __Einstellungen__
6. Tragen Sie dort die URL zu Ihrem Mietshop ein (diese erhalten Sie nach Installation des Shops per Mail).

Diese ist z.B. http://shop.mietshop.de/shop1410345442/cgi-bin/lshop.cgi
tragen Sie diese vollständig in das Textfeld „Shop URL“ ein.

== Frequently Asked Questions ==

= Was macht das Plugin / Widget, was ist der Zweck?" =
Das kostenlose Plugin ermöglicht es, Inhalte (eine Artikelbox mit Vorschaubild, Artikelname, Artikelpreis, Verfügbarkeit + Link zum Shop sowie eine Highlightartikel der Woche Box und eine Bestseller Box (mit den 10 bestverkauften Produkten) ganz einfach im Seiteneditor von Wordpress einzufügen.

Somit können Inhalte des Blogs/der Webseite mit den Artikeln aus dem Shop gemischt, Teaser aufbaut und somit mehr Besucher auf den Shop geleitet werden.

= Benötige ich zwingend einen Mietshop dazu, oder kann ich auch alles innerhalb von Wordpress verwalten? =

Ohne das Mietshop System ist es leider nicht möglich. Der Mietshop dient als Datenbasis für die Artikel. Dort werden diese angelegt und verwaltet. Der Shop läuft auch eigenständig, unter einer eigenen Domain.
Das Plugin ist dafür gedacht, Produkte im Blog anteasern zu können.

= Was kostet das Plugin? =
Das Plugin ist kostenlos.

= Was kostet der Mietshop? =
Der Mietshop ist die ersten 6 Monate völlig kostenlos. Erst dann entscheiden Sie sich für eine der günstigen kostenpflichtigen Varianten.

= Kann man auch einen Warenkorb innerhalb Wordpress darstellen? =
Das ist leider nicht möglich. Ggf. in einer der nachfolgenden Versionen des Plugins.
 
= Kann man mehrere Artikel in Wordpress dem Warenkorb hinzufügen? =
Das ist derzeit nicht möglich.

= Bei mir ist die Darstellung zerschossen/defekt, was kann ich tun? =
Stellen Sie zunächst sicher, dass Sie die Elemente mittels des „Warenkorb Buttons“ im Editor (auf Beiträgen, Seiten, usw..) eingefügt haben. Sollte es dennoch nicht funktionieren, kontaktieren Sie uns unter info@mietshop.de
 
= Ich möchte den gesamten Shop in meine Website/Wordpress einbinden, geht das? =
Ja! Bitte schreiben Sie uns dazu unter info@mietshop.de an.
